## 开发优先级说明

### 优先级评估标准

1. **高优先级**：
   - 国内主流平台，用户量大
   - 企业级应用场景
   - 有明确的 API 文档和 SDK

2. **中优先级**：
   - 国际主流平台
   - 有特殊功能需求
   - 社区需求较高

3. **低优先级**：
   - 小众平台
   - 特殊协议
   - 实验性功能
